# To-Do-App (ReactJS) - Currently in Development #

### Install dependencies using: `yarn install` or `npm install` ###
### Run app using: `yarn start` or `npm start` ###

## Included: ##
1. Home (Today's tasks) page
2. Profile Page
3. Task Viewing
4. Drawer
5. Calendar 
6. Study Timer
7. Suggested Tasks
